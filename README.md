# CODSOFT – Artificial Intelligence Internship

This repository contains all the tasks completed during the **CODSOFT Artificial Intelligence Internship**.

## Tasks Included

### Task 1 – Rule-Based Chatbot
- Developed a simple rule-based chatbot using Python
- Responds to user inputs based on predefined rules
- Concepts used: if-else conditions, string matching, basic conversation flow

### Task 2 – Tic Tac Toe AI
- Implemented an AI-based Tic Tac Toe game using Python
- AI uses the Minimax algorithm to make optimal decisions
- Human vs AI gameplay in command line

### Task 3 – Image Captioning
- Built an image captioning system using deep learning
- Generates textual descriptions for input images
- Demonstrates basic computer vision and NLP concepts

## Language Used
- Python

These tasks helped me understand the fundamentals of:
- Artificial Intelligence
- Game logic & decision making
- Natural Language Processing
- Computer Vision
